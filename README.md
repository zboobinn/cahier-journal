# Cahier journal

Application web personnelle pour préparer chaque journée de classe. Voir `CAHIER-DES-CHARGES.md` pour la spécification complète et `CLAUDE.md` pour les conventions de travail.

## Commandes

```bash
npm run dev       # développement (http://localhost:5173)
npm run build     # build de production (sortie : dist/)
npm run preview   # prévisualiser le build
```
